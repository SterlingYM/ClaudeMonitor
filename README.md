# ClaudeMonitor
A dashboard to monitor all active agents on your local machine

### Why I made this tool:
I had way too many agents running. Multiple agents running in parallel within projects, many VSCode sessions, many terminal windows, all waiting for my inputs, requesting permissions, stuck in loop, etc. etc...
So I wanted to make my own HQ to monitor all of my agents doing/not doing work.
